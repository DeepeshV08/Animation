# Animation Learning Journey

## About
This repository contains my animation learning exercises, notes, projects, and experiments. The goal is to build a strong foundation in animation principles and gradually create more advanced animations.

## Learning Goals
- Understand the 12 Principles of Animation
- Learn timing and spacing
- Practice character movement
- Create smooth transitions and effects
- Build animation projects from beginner to advanced level

## Topics Covered

### Beginner
- Animation basics
- Keyframes
- Timing
- Spacing
- Ease In & Ease Out

### Intermediate
- Squash and Stretch
- Anticipation
- Follow Through
- Secondary Action
- Motion Paths

### Advanced
- Character Animation
- Lip Sync
- Walk Cycles
- Physics-Based Animation
- Visual Effects
